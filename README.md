# Aplikacja web GIS – wybór miejsca zamieszkania w Poznaniu

## Opis projektu
Interaktywna aplikacja webowa wspomagająca wybór miejsca zamieszkania w Poznaniu na podstawie wielokryterialnej analizy przestrzennej.

## Stack
- Python
- Dash
- Dash Bootstrap Components
- Dash Leaflet
- GeoPandas

## Dane
- BDOT10k
- OpenStreetMap

## Uruchomienie lokalne
1. Zainstaluj biblioteki:
   pip install -r requirements.txt

2. Uruchom aplikację:
   python app.py
